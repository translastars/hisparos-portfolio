🔌 AJAX-Ready API Integration

Connect this frontend to your backend or a third-party terminology API. Below is the expected endpoint structure for developers.

// GET /api/terms — Search the terminology database
// Supports query params: q, lang, domain, page, limit

GET /api/v1/terms?q=habeas+corpus&lang=en-es&domain=Legal&page=1&limit=20

// Response format (JSON):
{
  "total": 5,
  "page": 1,
  "terms": [
    {
      "source_term": "habeas corpus",
      "target_term": "hábeas corpus",
      "source_lang": "en",
      "target_lang": "es",
      "domain": "Legal",
      "context": "The right to ...",
      "usage_example": "..."
    }
  ]
}

// JavaScript frontend example:
const res = await fetch(`/api/v1/terms?q=${query}&lang=${lang}&domain=${domain}`);
const data = await res.json();
renderTerms(data.terms);
